# MILO Pet Shop

Landing page moderna para MILO Pet Shop, construida con Astro y Tailwind CSS.

## Tecnologías

- **Astro** — Framework web estático
- **Tailwind CSS** — Estilos utilitarios
- **Iconify** — Iconos bajo demanda
- **Font Awesome** — Iconos de redes sociales

## Secciones

- Hero con watermark "MILO"
- Nosotros
- Productos (6 categorías con imágenes reales)
- Galería
- Contacto (Instagram + WhatsApp + dirección)
- WhatsApp flotante CTA
- Footer

## Desarrollo

```bash
npm install
npm run dev      # Servidor local
npm run build    # Build estático en dist/
```

## Deploy en Vercel

1. Conectá tu repositorio de GitHub en [vercel.com](https://vercel.com)
2. Vercel detecta automáticamente Astro — no requiere configuración adicional
3. Hacé click en **Deploy**

También podés deployar desde la CLI:

```bash
npm i -g vercel
vercel
```
